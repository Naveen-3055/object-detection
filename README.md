Snap & Speak is a React-based AI application that uses your device camera to capture an image, identifies the main object using Google Gemini Vision API, and speaks the result aloud using browser Text-to-Speech.
This project is fully frontend-only and demonstrates real-time AI vision integration in the browser.

 Features
 
  1.Live camera access using Web APIs
  
  2.Capture image from video stream
  
  3.Object recognition using Gemini 2.5 Flash
  
  4.Simple object explanation in plain English
  
  5.Automatic voice narration (Text-to-Speech)
  
  6.Fast and lightweight (no backend required)
